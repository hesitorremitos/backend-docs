La programación del lado del servidor es el proceso de crear sitios o aplicaciones web que se ejecutan en un servidor y que son entregados al navegador del usuario.
- **Sitios web dinamicos** Los sitios web son generados de manera dinámica en el servidor, es decir, el servidor genera el contenido HTML y lo envía al navegador del usuario.
- **Aplicaciones web** Las aplicaciones web son programas que se ejecutan en el servidor y que son entregados al navegador del usuario. Estas aplicaciones pueden ser tan simples como un formulario de contacto o tan complejas como un sistema de gestión de contenidos.

## ¿Cómo funciona?
Para que el servidor pueda entregar contenido al navegador del usuario, se necesita un **lenguaje de programación** que pueda generar contenido HTML, que se encargue de toda la logica y el dinaismo de la pagina web, como almacenar datos en una base de datos, generar PDF, editar imagenes, etc.

<Steps>

1. El navegador del usuario solicita una página web al servidor(url).
2. El servidor recibe la solicitud y ejecuta el código del lado del servidor.
3. El servidor genera el contenido HTML
4. El servidor envía el contenido HTML al navegador del usuario.
5.  El navegador del usuario muestra la página web al usuario.

</Steps>

Existen muchos lenguajes de programación que se pueden utilizar para programar del lado del servidor, algunos de los más populares son:
- **PHP** es un lenguaje de programación de código abierto que se utiliza para crear sitios web dinámicos.
- **Python** es un lenguaje de programación de alto nivel que se utiliza para crear aplicaciones web.
- **Node.js** es un entorno de ejecución de JavaScript que se utiliza para crear aplicaciones web.
- **Ruby** es un lenguaje de programación de código abierto que se utiliza para crear aplicaciones web.


## PHP
PHP nacio con la idea de dar  **vida** a las paginas web(HTML), para que estas dejen de ser simples **paginas estaticas** y se conviertan en paginas dinamicas.

Algunas de sus caracteristicas son:
- **Codigo abierto**, lo que significa que es gratuito y que cualquiera puede utilizarlo.
- **Amplio soporte**, lo que significa que hay una gran cantidad de documentación y tutoriales disponibles.
- **Recursos de la comunidad**, lo que significa que hay una gran cantidad de bibliotecas y frameworks disponibles.
- **Multiproposito**, lo que significa que se puede utilizar para crear una amplia variedad de aplicaciones web.
- **Multiparadigma**, lo que significa que se puede utilizar para crear aplicaciones web de diferentes maneras. 
- **El mas usado**, es el lenguaje mas usado en cuando a desarollo web se refiere.
