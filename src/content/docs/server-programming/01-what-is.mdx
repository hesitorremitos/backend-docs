La programación del lado del servidor es el proceso de crear sitios o aplicaciones web que se ejecutan en un servidor y que son entregados al navegador del usuario.

```d2 pad=30
shape: sequence_diagram
client: CLIENTE
server: SERVER

# (*->*).style.animated: true
client->server: Peticion de recursos
server->client: Respuesta

```

Para hacer esto posible se requiere de un **servidor web**, que es un programa que se ejecuta en un servidor y que se encarga de recibir las solicitudes de los navegadores de los usuarios y de enviarles el contenido HTML de las páginas web.

Los programas mas comunes para crear un servidor web son:
- **Apache2**, es el mas usado en el mundo.
- **Nginx**, es el segundo mas usado en el mundo.
- **IIS**, es el servidor web de Microsoft, funciona solo en Windows.
- Otros que se apoyan a un lenguaje de programación como Node.js, Python, Ruby, etc.



## Servidor web(Apache2)
En el curso de programación del lado del servidor, se utilizara Apache2 como servidor web, ya que es el mas usado en el mundo y es muy facil de configurar.

Algunas de sus caracteristicas son:
- **Codigo abierto**, lo que significa que es gratuito y que cualquiera puede utilizarlo.
- **Amplio soporte**, lo que significa que hay una gran cantidad de documentación y tutoriales disponibles.
- **Recursos de la comunidad**, lo que significa que hay una gran cantidad de bibliotecas y frameworks disponibles.
- **Multiproposito**, lo que significa que se puede utilizar para crear una amplia variedad de aplicaciones web.


## Otras Herramientas
Aparte de un *servidor web se requieren elementos adicionales para poder ejecutar aplicaciones web, como ser:
- **Base de datos**, comunmente el backend de una aplicacion web necesita almacenar datos en una base de datos.
- **Lenguaje de programación**, es el encargado de generar el contenido HTML **dinamico** que se envia al navegador del usuario.
- **Framework**, en aplicaciones web complejas se necesita un framework que facilite el desarrollo de la aplicacion.
- **Herramientas de desarrollo**, para poder desarrollar aplicaciones web se necesita un editor de codigo, un navegador web, etc.
- Otros como **servidor de correo, servidor de archivos, etc**.

Como se puede ver, la programación del lado del servidor es un proceso complejo que requiere de una gran cantidad de herramientas y conocimientos para poder llevarlo a cabo.

Debido a esta dificultad nacen las aplicaciones como **XAMPP, WAMP, LAMP, MAMP**, que son paquetes que incluyen todo lo necesario para poder desarrollar aplicaciones web en un solo lugar.
