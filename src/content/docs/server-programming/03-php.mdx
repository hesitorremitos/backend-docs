## Introducción a PHP
PHP es un lenguaje que usa la **Programacion procedural** principalmente, ya que la mayoria de sus funciones son de este tipo. Sin embargo, también soporta la **Programación orientada a objetos** (OOP) y la **Programación funcional**.
:::nota
La programacion procedural es un paradigma de programación que se basa en la idea de dividir un programa en procedimientos o funciones.  Entonces los datos primitivos no poseen metodos ni propiedades, y para tratar con ellos debemos usar funciones integradas del lenguaje.

## Fundamentos

### Sintaxis Básica
`echo` y `print` son constructores del lenguaje que permiten mostrar una o más cadenas de texto. La principal diferencia es que `print` solo puede tomar un argumento y siempre devuelve 1.

:::tip
Tambien es posible usar las secuencias de escape `\n` y `\t` para crear saltos de línea y tabulaciones respectivamente.
:::


```php
<?php
echo "Hola Mundo";
echo "Hola Mundo\n"; // con salto de línea
```
```php
<?php
print("Hola Mundo");
print "Hola Mundo"; // También válido
```

### Variables
Las variables en PHP son contenedores para almacenar información. Se identifican siempre con el símbolo `$` seguido del nombre de la variable.

```php
// Variables y ejemplos de uso
$nombre = "Juan" // String
echo $nombre; // Salida: Hola Juan

$edad = 25; // Integer
echo $edad; // Salida: 30

$precio = 19.99; // Float 
echo $precio; // Salida: 39.98

$activo = true; // Boolean 
echo $activo; // Salida: 1

$datos = null; // Null - Sin valor
echo $datos; // Salida: (nada)
```

#### Obtener el Tipo de Dato
Para poder obtener el tipo de dato de una variable, se puede usar la función `gettype()`. También se pueden usar funciones como `is_string()`, `is_int()`, `is_float()`, para verificar el tipo.

```php
// Ejemplo de tipos de datos
$nombre = "Juan";
$edad = 25;
$precio = 19.99;
$activo = true;
$datos = null;
echo gettype($nombre); // string
echo gettype($edad); // integer
echo gettype($precio); // double
echo gettype($activo); // boolean
echo gettype($datos); // NULL
```

### Concatenación
La concatenación en PHP se realiza con el operador `.` (punto). 

```php
// Concatenación de cadenas
$nombre = "Juan";
$apellido = "Pérez";
echo $nombre . " " . $apellido; //Nombre completo
print("Hola " . $nombre . "\n"); // Hola Juan
```
También se puede usar la interpolación de variables dentro de comillas dobles.

```php
echo "Hola, $nombre $apellido\n"; // Hola, Juan Pérez
print("Hola, $nombre $apellido\n"); // Hola, Juan Pérez
```

Otra forma de concatenar es usando las comillas simples, pero en este caso no se interpretan las variables.

:::caution
Las comillas simples no interpretan variables, por lo que se deben usar comillas dobles para la interpolación.
```php
echo 'Hola, $nombre $apellido\n'; // Esto no interpreta las variables
```
:::

### Operadores
Los operadores son iguales a muchos otros lenguajes de programación.

#### Operadores Aritméticos
Los operadores aritméticos son los que se utilizan para realizar operaciones matemáticas.

```php	
// Ejemplo de operadores aritméticos
$suma = 5 + 3;      // Suma
$resta = 10 - 2;    // Resta
$multiplicacion = 4 * 2; // Multiplicación
$division = 10 / 2; // División
```
#### Operadores de Comparación
Los operadores de comparación se utilizan para comparar dos valores y devuelven un valor booleano (true o false).

| Operador | Descripción |
|----------|-------------|
| `==`     | Igual a    |
| `<`    | Menor que  |
| `>`    | Mayor que  |
| `<=`   | Menor o igual que |
| `>=`   | Mayor o igual que |
| `!=`   | Diferente de |
| `===`  | Igual y del mismo tipo |
| NULL | Se intepreta como falso |
| 1 | Se intepreta como verdadero |
| 0 | Se intepreta como falso |

```php	
1 == 1 // true
1 > 2 // false
1 < 2 // true
1 != 2 // true
/ etc...
```

#### Operadores Lógicos
Los operadores lógicos se utilizan para combinar expresiones booleanas.
| Operador | Descripción |
|----------|-------------|
| `&&`     | Y lógico    |
| `||`     | O lógico    |
| `!`      | No lógico   |

```php
// Ejemplo de operadores lógicos
1 == 1 && 2 == 2 // true AND true = true
1 == 1 || 2 == 3 // true OR false = true
1 == 1 && 2 == 3 // true AND false = false
! (1 == 1) // false
```

#### Operadores de Asignación
Los operadores de asignación se utilizan para asignar valores a variables.

```php
// Ejemplo de operadores de asignación
$numero = 5; // Asignación simple
$numero += 2; // Asignación con suma
$numero -= 3; // Asignación con resta
$numero *= 2; // Asignación con multiplicación
$numero /= 2; // Asignación con división
$numero %= 2; // Asignación con módulo
```

## Arrays
Los arrays son estructuras de datos que permiten almacenar múltiples valores en una sola variable. PHP soporta arrays indexados y asociativos.

### Arrays Indexados
Podriamos decir que son los arrays normales, donde cada elemento tiene un índice numérico. Para acceder a un elemento, se usa el índice entre corchetes `[]`.

```php
// Ejemplo de array indexado
$frutas = ['manzana', 'pera', 'uva'];
echo $frutas[0]; // Salida: manzana
echo $frutas[1]; // Salida: pera
echo $frutas[2]; // Salida: uva
```

:::note
A diferente de otros lenguajes, PHP no requiere declarar el tipo de array. Un array puede contener diferentes tipos de datos.
```php
// Array con diferentes tipos de datos
$datos = ['Juan', 25, 19.99, true];
echo $datos[0]; // Salida: Juan
echo $datos[1]; // Salida: 25
echo $datos[2]; // Salida: 19.99
echo $datos[3]; // Salida: 1 (true)
```
:::

### Arrays Asociativos
Los arrays asociativos son similares a los **dictcionarios** en otros lenguajes, donde cada elemento tiene una clave asociada. Para acceder a un elemento, se usa la clave entre comillas.

```php
// Ejemplo de array asociativo
$persona = [
    'nombre' => 'Juan',
    'edad' => 25,
    'ciudad' => 'Madrid'
    'activo' => true
];
echo $persona['nombre']; // Salida: Juan
echo $persona['edad']; // Salida: 25
echo $persona['ciudad']; // Salida: Madrid
```

### Funciones de Arrays
```php
// Funciones útiles para arrays
$frutas = ['manzana', 'pera', 'uva'];
$frutas[] = 'naranja'; // Agregar elemento al final
array_push($frutas, 'kiwi'); // Hace lo mismo
array_pop($frutas); // Elimina el último elemento
array_shift($frutas); // Elimina el primer elemento
``` 



## Funciones
:::tip[Definición]
Las funciones son bloques de código reutilizables que realizan una tarea específica y pueden recibir parámetros y devolver valores.
:::

```php
// Ejemplo de función con tipos y valor por defecto
function calcularPrecioTotal(float $precio, float $impuesto = 0.21): float {
    return $precio * (1 + $impuesto);
}

// Uso de la función
echo calcularPrecioTotal(100);      // 121
echo calcularPrecioTotal(100, 0.1); // 110
```
