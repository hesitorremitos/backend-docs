PHP es un lenguaje de programación que fue creado principalmente para añadir funcionalidades a las páginas web. Es un lenguaje de programación de propósito general, que se puede utilizar para crear aplicaciones web, aplicaciones de escritorio, aplicaciones móviles, etc.

Algunas de sus características son:
- **Codigo abierto**, lo que significa que es gratuito y que cualquiera puede utilizarlo.
- **Amplio soporte**, lo que significa que hay una gran cantidad de documentación y tutoriales disponibles.
- **Recursos de la comunidad**, lo que significa que hay una gran cantidad de bibliotecas y frameworks disponibles.
- **Multiparadigma**, lo que significa que significa que puedes programar en PHP de diferentes maneras, como ser **orientado a objetos**, **procedural**, **funcional**, etc.


## Formas de impresion en PHP
En PHP hay varias formas de imprimir texto en la pantalla

La mas comun es con la funcion `echo`

:::note
- Para escribir codigo php, el archivo debe tener la extension `.php`
- Al inicio de un archivo php se debe abrir con `<?php` y al final se debe cerrar con `?>`
  - Si el archivo solo contiene codigo php se recomienda no cerrar el archivo con `?>`
:::

```php
<?php
echo "Hola Mundo";
```
