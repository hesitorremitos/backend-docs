## Uso de PHP en el Navegador
Apache tiene un módulo llamado `mod_php` que permite ejecutar código PHP directamente en el servidor. Para usar PHP en el navegador, debes tener un servidor web instalado y configurado.

:::nota
Por defecto Laragon viene con Apache y PHP preinstalados. Solo necesitas crear un nuevo proyecto y abrirlo en el navegador.
:::
Al igual que los archivos HTML, los archivos PHP deben estar en la carpeta raíz del servidor web. Por defecto, esta carpeta es `www` o `htdocs` dependiendo del servidor que uses.
