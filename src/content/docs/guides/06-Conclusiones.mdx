Todo lo visto anteriormente en [Fundamentos web](/guides/01-introduccion) es solo la punta del iceberg en el desarollo web **Frontend**.

Como se dijo en el anterior curso el desarollo web se divide en dos partes, **Frontend** y **Backend**. En este curso se vio la parte de **Frontend** que es la parte visual de una pagina web, la parte que interactua con el usuario.

El **Frontend** es un buen punto de entrada para conocer como fuciona lo que esta detras de este(**El Backend**), ya que se puede ver los datos que se envian, y ahora veremos como se reciben esos datos y los procesaremos(Almacenar en una base de datos)


:::note
En el Temario se vio lo minimo necesario para poder crear una pagina web, mas adelante veremos mas elementos del **Frontend** y se pide que el estudiante siga investigando por su cuenta.
:::
