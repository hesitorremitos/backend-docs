Ya que hemos creado la estructura básica de nuestro documento HTML, es hora de agregar estilos para que se vea más atractivo. En este tutorial, nos enfocaremos en los estilos CSS básicos para darle un aspecto más agradable a nuestro formulario de inicio de sesión.
Usaremos el anterior documento HTML(Login) como base para agregar estilos.


## Reglas CSS
Tambien conocida como **sintaxis** de CSS. Se componente principalmente de 2 partes: Los **selectores** y las **declaraciones**.

Los selectores son la parte de la regla CSS que indica a qué elementos se aplicarán los estilos. Pueden ser de diferentes tipos, como selectores de etiqueta, selectores de clase, selectores de ID, etc.

### Selectores de etiqueta
Selecciona todas las etiquetas con el nombre especificado como  `h1`, `div`,`br`, `p`, etc.

```html "h1" "div" "br" "p"
<h1>Titulo</h1>
<div>Contenido</div>
<br>
<p>Párrafo</p>
```


### Selector de clase
Selecciona todas las etiquetas con la clase especificada como `.[clase]`. del atributo `class` de las etiquetas HTML.

```html "contenedor" "formulario"
<div class="contenedor formulario">
    <h1>Titulo</h1>
</div>
```
De lo anterior se dice que el contenedor tiene la clase `.formulario` y `.contenedor`.
El `.`(punto) indica que es una clase.

### Selector de ID
Selecciona una etiqueta con el ID especificado como `#[id]`. del atributo `id` de las etiquetas HTML.

```html "formulario" "title"
<div id="formulario">
    <h1 id="title">Titulo</h1>
</div>
```
De lo anterior se dice que el contenedor tiene el id `#formulario`, y el titulo tiene el id `#title`.
El `#`(numeral) indica que es un ID.


### Declaraciones
Las declaraciones son la parte de la regla CSS que define los estilos que se aplicarán a los elementos seleccionados. Se componen de una propiedad y un valor separados por dos puntos `:`.
```css "propiedad : valor;"
h1 {
propiedad : valor;
}
```

## Agregando estilos CSS
Para agregar estilos a nuestro documento HTML, usaremos la etiqueta `<style>` dentro de la etiqueta `<head>`. Esta etiqueta se usa para definir estilos en línea, que se aplican directamente a las etiquetas HTML.

```html {4-6}
<head>
    <title>Mi primer login</title>
    <link rel="icon" href="favicon.ico">
    <style>
        /* Estilos CSS */
    </style>
</head>
```

## Agregando estilos a etiquetas HTML
Como mencionamos, usaremos en **login** anteioremente creado como base para agregar estilos.
Dentro de la etiqueta `<style>`, agregaremos los estilos CSS para darle un aspecto más atractivo a nuestro formulario de inicio de sesión.

Agregaremos un fondo a nuestro formulario, siguiendo los siquintes pasos:

<Steps>

1. Identificamos la etiqueta a estilizar, en este caso `<form>`. y despues abrimos y cerramos las llaves `{}`.
    ```css
    fieldset {
        /* Estilos para el fieldset */
    }
    ```

2. Dentro de las llaves, agregamos las propiedades y valores CSS para darle estilo al fieldsetulario. En este caso, agregaremos un color de fondo.

    ```css
    fieldset {
        background-color: red;
    }
    ```
3. Guardamos los cambios y recargamos la página para ver el efecto de los estilos.

4. Modificamos el color a uno mas atractivo usando el valor hexadecimal `#RRGGBB` o el nombre del color.
    :::note
    Tambien puede usar los diferentes tipos de colores como: 
    - `rgb(255, 0, 0)` para rojo.(Red, Green, Blue)
    - `rgba(255, 0, 0, 0.5)` para rojo con transparencia.(Red, Green, Blue, Alpha)
    - `hsl(0, 100%, 50%)` para rojo.(Hue, Saturation, Lightness)
    - `hsla(0, 100%, 50%, 0.5)` para rojo con transparencia.(Hue, Saturation, Lightness, Alpha)

    Estos son otros tipos de colores que puede usar en CSS.
    :::

    ```css
    fieldset {
        background-color: #f0f0f0;
    }
    ```
</Steps>

## Agregando estilos a clases y IDs
Además de agregar estilos a las etiquetas HTML, también podemos agregar estilos a clases y IDs. En nuestro formulario de inicio de sesión, para lo cual añadiremos algunas clases y IDs a las etiquetas HTML.

```html "class="form""  "class="contenedor"" "class="campo"" "id="username"" "id="password"" ins={8-12}
<!DOCTYPE html>
<html lang="es">
    <head>
        <title>Mi primer login</title>
        <link rel="icon" href="favicon.ico">
    </head>
    <!--Estilos -->
    <style>
        fieldset {
            color: #f0f0f0;
        }
    </style>

    <body>
        <form clase="login">
            <fieldset>
                <legend>Inicio de sesión</legend>
                <div class="campo"> 
                    <!-- Campo usuario -->
                    <label for="username">Usuario:</label>
                    <input type="text" id="username" placeholder="Ingrese su usuario">
                </div>
                <br>
                <div>
                    <!-- Campo contraseña -->
                    <label for="password">Contraseña:</label>
                    <input type="password" id="password" placeholder="Ingrese su contraseña">
                </div>
                <br>
                <button class="boton"  type="submit">Iniciar sesión</button>
            </fieldset>
        </form>
    </body>
</html>
```

Continuamos con la etiqueta de estilo `<style>` y agregamos estilos para las clases y los IDs que hemos agregado a las etiquetas HTML.

```html
<style>
    fieldset{
        background-color: #f0f0f0;
    }
    .login{
        width: 30%;
        margin: auto;
    }

    /*Estilos para los campos de texto */
    #username, #password{
        width: 100%;
    }

    /*Estilos para las etiquetas */
    label{
        font-family: sans-serif;
    }
    
    /*Estilos para el botón */
    .boton{
        width: 100%;
        background-color: darkblue;
        color: white;
    }
</style>
```

Ahora tenemos un login mucho mas atractivo y con estilos CSS básicos.

:::caution
Cabe mencionar que estos estilos no afectan en el funcionamiento del formulario, solo son para darle un aspecto más atractivo.
:::
